Типы отношений:
Ассоциация - тип отношений когда объекты ссылаются друг на друга, независимо друг от друга (не всегда), наличие связей между объектами
    Кратность связей (1 к М, М к М, 1 к 1, М к 1)
Композиция - тип отношения при котором один объект может принадлежать только другому объекту и никому иначе.
    Основной объекта полностью контролирует жизненный цикл объектов от которых он зависит.
    Сильная ссылка
Агрегация - тип отношений когда один объект является частью другого
    Похожа на композицию только зависимости создаются вне основного объекта
