Сложность:
O(n) = n^2 | n * n
Примеры сортировок:
    пузырьком - BubbleSort
    вставкой - InsertionSort
    выбором - SelectionSort

Сложность:
O(n) = n*log(n)
Примеры сортировок:
    быстрая сортировка - QuickSort
    сортировка слиянием - MergeSort

Линейная сложность:
O(n) = n
Примеры сортировок:
    Сортировка подсчетом - CountSort
